Connector for Salesforce Lightning Platform

overview

A set of 75 Activities and 5 wizards that allow you to do a very fast implementation for Salesforce Lightning Platform.

For this activity there are around 20 movies on youtube:
https://www.youtube.com/channel/UCiSb0WLTgB23jT0E6NEuu7Q

The Salesforce Activity package allows you to perform various operations such as adding, deleting, updating, or getting (ADD, DELETE, UPDATE or GET) on the main entities of Salesforce. The main entities include ACCOUNT, ASSET, CASE, CONTACT, LEAD, CAMPING, FILE, OPPORTUNITY, TASK,PRICEBOOK,PRODUCT,QUOTE. For the other tables in Salesforce and for the custom ones, the package contains Add, Update and Delete activities that can be used without any restriction by name or structure.
